# Student Performance Analysis System

> This project is developed as part of the **Knowvation Learning Program**.

## Problem Statement

Educational institutions often face challenges in analyzing student academic performance efficiently due to the large amount of data generated from various student activities and records. Traditional methods of evaluating student performance may not provide clear insights into the factors that significantly influence academic success.

The purpose of this project is to build a **Student Performance Analysis System** that uses data analysis and machine learning techniques to study the relationship between different student-related factors and their final academic performance.

The system analyzes attributes such as:
- Hours Studied
- Attendance Rate
- Previous Scores
- Participation in Extracurricular Activities
- Sleep Hours

Using these factors, the project aims to:
- Analyze student performance patterns
- Visualize relationships between academic factors and final scores
- Predict student final scores using machine learning algorithms
- Evaluate prediction accuracy using performance metrics
- Identify the most influential features affecting student performance

The project applies data preprocessing, exploratory data analysis (EDA), visualization techniques, and a Random Forest Regression model to generate meaningful insights and predictions from the dataset.

## Project Objective

The main objective of this project is to develop a machine learning-based system capable of analyzing and predicting student academic performance using multiple educational and behavioral factors.
